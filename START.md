# 🚀 Quick Start Guide

## Run the Website

```bash
cd cnri-website
npm run dev
```

Then open: **http://localhost:3000**

## What You'll See

✅ **8 Complete Pages:**
- Home (Hero, Focus Areas, Impact Stats, Activities)
- About (Mission, Vision, Overview)
- Focus Areas (5 detailed areas)
- Programs (10 programs with descriptions)
- Conference (Food for Education Policy Conference)
- Team (Staff, Advisory Board, Interns)
- Partners (5 strategic partners)
- Contact (Form, Info, Newsletter)

✅ **Modern Design:**
- Green/Orange/White color scheme
- Smooth animations and hover effects
- Fully responsive mobile design
- Professional research institute + NGO aesthetic

✅ **Ready for Production:**
- TypeScript for type safety
- Tailwind CSS for styling
- Component-based architecture
- SEO-friendly structure
